# VisuallyHidden

Hides content from the screen in an accessible way.

`VisuallyHidden` hides content from visual client, but keep it readable for screen readers. It is useful in certain scenarios as an alternative to traditional labelling with aria-label or aria-labelledby and it is the logical opposite of the aria-hidden attribute.
