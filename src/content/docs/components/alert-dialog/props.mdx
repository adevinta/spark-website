## AlertDialog

Contains all the parts of an alert dialog.

## AlertDialog.Trigger

A button that opens the dialog.

<ComponentProps docgen={docgen["AlertDialog.Trigger"]} />

## AlertDialog.Portal

When used, portals your overlay and content parts into the body.

<ComponentProps docgen={docgen["AlertDialog.Portal"]} />

## AlertDialog.Overlay

A layer that covers the inert portion of the view when the dialog is open.

<ComponentProps docgen={docgen["AlertDialog.Overlay"]} />

## AlertDialog.Content

Contains content to be rendered when the dialog is open.

<ComponentProps docgen={docgen["AlertDialog.Content"]} />

## AlertDialog.Header

To use inside of AlertDialog.Content. Sticky header inside the dialog.

<ComponentProps docgen={docgen["AlertDialog.Header"]} />

## AlertDialog.Body

To use inside of AlertDialog.Content. Main content of the dialog. Becomes scrollable when content is overflowing the screen.

<ComponentProps docgen={docgen["AlertDialog.Body"]} />

## AlertDialog.Footer

To use inside of AlertDialog.Content. Sticky footer to be rendered in the open dialog. Use it to render CTAs and make them accessible in every condition.

<ComponentProps docgen={docgen["AlertDialog.Footer"]} />

## AlertDialog.Title

An accessible name to be announced when the dialog is opened.

<ComponentProps docgen={docgen["AlertDialog.Title"]} />

## AlertDialog.Description

An accessible description to be announced when the dialog is opened.

<ComponentProps docgen={docgen["AlertDialog.Description"]} />

## AlertDialog.Cancel

A button that closes the dialog. This button should be distinguished visually from AlertDialog.Action buttons.

<ComponentProps docgen={docgen["AlertDialog.Cancel"]} />

## AlertDialog.Action

A button that closes the dialog. These buttons should be distinguished visually from the AlertDialog.Cancel button.

<ComponentProps docgen={docgen["AlertDialog.Action"]} />
